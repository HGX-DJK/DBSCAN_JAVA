DBSCAN 空间聚类分析工具

基于密度的空间聚类算法实现

使用说明

  1. 选择或上传示例 CSV 文件
  2. 设置经纬度字段名称（默认: lng, lat）
  3. 调整聚类参数并执行分析
  4. 结果将显示在地图上
参数调整建议：
  • 距离阈值 (epsilon)：单位为千米，值越小，聚类越密集（建议 0.1-5 千米）
  • 最小聚类大小：值越小，聚类越多，噪声点越少（建议 2-10）
  • 最小样本数：值越小，更多点会被包含在聚类中（建议等于或小于最小聚类大小）
  • 如果噪声点太多，尝试减小距离阈值或最小样本数
